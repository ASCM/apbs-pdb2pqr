2.2. Binary installation

We currently offer RPM-format binaries for the RedHat Linux platform on a variety of architectures as well as command line binaries for WinXP and Mac OS X. Binaries can be downloaded from the APBS download page. For all other systems, please install from source on your particular platform and feel free to contact the APBS users mailing list for more help.

In order to use the APBS binaries you will also need MALOC binaries - these can be obtained from http://www.scicomp.ucsd.edu/~mholst/codes/maloc/index.html.
